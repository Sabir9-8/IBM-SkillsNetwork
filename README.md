# 📂 IBM Full Stack Web Dev — Learning Projects

This repository contains hands-on projects I'm building as part of my **HTML, CSS, and JavaScript** learning journey through IBM SkillsNetwork.

Projects are added continuously as I progress through the curriculum. Each folder is a standalone mini-project focused on a specific concept or feature.

---

## 🗂️ Projects

| Folder | Description | Concepts Covered |
|---|---|---|
| `CalculateArea` | Calculates area of different shapes based on user input | DOM manipulation, input handling, conditionals |
| `Scope` | Demonstrates the difference between `var`, `let`, and `const` | Variable scope, hoisting, block scope |
| `employeeDetails` | Tracks and displays employee information | Objects, arrays, DOM rendering |
| `productSurvey` | A survey form that collects and displays product feedback | Forms, event listeners, data handling |
| `restaurantMenu` | Displays a dynamic restaurant menu | Array methods, template literals, DOM |

> More projects will be added as I continue through the course.

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript (ES6+)

## 🎯 Goal

Learn and apply core web development concepts through small, focused projects — one concept at a time.
